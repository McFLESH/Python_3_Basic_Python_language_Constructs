**Логический тип данных (bool)**


Значения
- True — логическая истина (математически 1)
- False — логическая ложь (математически 0)


Основные операторы
- and — логическое И (возвращает True, если оба операнда истинны)
- or — логическое ИЛИ (возвращает True, если хотя бы один операнд истинен)
- not — логическое НЕ (инвертирует значение, возвращает противоположное)


Конструкция if-else
- if условие1:
-     pass
- elif условие2:
-     pass
- else:
-     pass


Оператор for наиболее часто используется для перебора итерируемых объектов. Простой вариант использования – это перемещение по элементам объекта с помощью оператора in. Здесь он получает элементы итерируемого объекта. Конструкция завершается двоеточием, после чего обязательно должен идти внутренний блок кода, который будет выполняться заданное число раз. Пример кода:
- l1 = [1,"2",'df',4]
- for i in l1:
-     print(i)


Переменная i изменяется внутри цикла for, а количество итераций цикла зависит от количества элементов в списке.
Бывают случаи, когда вы не хотите проходить непосредственно по элементам, тогда можно использовать специальную функцию range, которая создает итерируемый объект в необходимом диапазоне. Функция range имеет три ключевых параметра - START:STOP:STEP. Она генерирует объекты от номера START, до STOP (НЕ включая его), с шагом STEP. По умолчанию START = 0, STOP – обязательный аргумент, который нужно указать в скобках, STEP = 1. range может использоваться как для контроля количества итераций или генерации простых значений, так и для прохождения по элементам списка. 


Иногда возникает потребность повторять фрагмент кода, пока выполняется условие. В таком случае более удобно будет использовать цикл while. Синтаксически конструкция пишется следующим образом: «while условие двоеточие», после чего идет блок повторяемого кода. Приведем пример:
- cost1 = 60000
- my_money = int(input("Размер бюджета: "))
- while my_money >= cost1:
-     print(' Заказывайте рекламу.')
-     my_money = my_money - cost1


break – оператор выхода из цикла. При вызове этого оператора цикл сразу завершит работу. Этот оператор используется, чтобы реализовать специфические условия выхода из цикла.


Оператор continue используется, чтобы пропустить одно текущее действие цикла после вызова оператора, после чего цикл продолжится в обычном режиме.

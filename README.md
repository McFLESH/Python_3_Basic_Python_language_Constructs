**Логический тип данных (bool)**


Значения
- True — логическая истина (математически 1)
- False — логическая ложь (математически 0)


Основные операторы
- and — логическое И (возвращает True, если оба операнда истинны)
- or — логическое ИЛИ (возвращает True, если хотя бы один операнд истинен)
- not — логическое НЕ (инвертирует значение, возвращает противоположное)


Конструкция if-else
- if условие1:
-     pass
- elif условие2:
-     pass
- else:
-     pass


Оператор for наиболее часто используется для перебора итерируемых объектов. Простой вариант использования – это перемещение по элементам объекта с помощью оператора in. Здесь он получает элементы итерируемого объекта. Конструкция завершается двоеточием, после чего обязательно должен идти внутренний блок кода, который будет выполняться заданное число раз. Пример кода:
- l1 = [1,"2",'df',4]
- for i in l1:
-     print(i)
